# Orivah

Orivah is a full-stack cashflow and invoice risk assistant for freelancers and small businesses.

The webapp is intended help users manage clients, track invoices, view dashboard insights, and identify cashflow risk from unpaid or overdue invoices.

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python, SQLAlchemy
- Database: SQLite for local development
- Authentication: JWT

## Current Features

- Landing page
- Login flow
- JWT authentication
- Protected API routes
- Client management
- Invoice management
- Dashboard summary connected to backend

## Work in Progress

- Invoice view/edit actions
- Client detail page
- Cashflow forecasting
- Risk scoring
- Reminder email generator
- UI polish and responsive improvements

## Project Status

This project is currently in MVP development.
